安装仅需 30 秒，无需复杂配置。支持 npm、Homebrew、直接下载。

# 或使用 Homebrew (macOS)
$ brew install claude-code

# 进入项目，开始对话
$ cd my-project && claude

常见问题

你关心的，都在这里

Claude Code 如何收费？
Claude Code 本身免费使用，通过 Anthropic API 按 token 计费。你也可以接入第三方 API 提供商（如 MiniMax、Kimi、DeepSeek）降低成本。建议日常使用 Sonnet 模型，复杂任务切换到 Opus。
支持哪些编程语言？
Claude Code 支持所有主流编程语言，包括但不限于：TypeScript、Python、Rust、Go、Java、C/C++、Ruby、Swift、Kotlin 等。它对前端框架（React、Vue、Next.js）和后端框架（FastAPI、Express、Spring）有特别深的理解。
我的代码安全吗？
安全是 Anthropic 的第一优先级。通过 API 使用时，你的代码不会用于模型训练。Claude Code 支持 SOC 2 合规，所有数据传输加密。权限分级系统确保敏感操作（如删除文件、执行危险命令）需要用户显式授权。
与 GitHub Copilot 有什么区别？
Claude Code 是终端原生的自主编程助手，能主动探索项目、执行多步骤任务、运行测试并修复错误；而 Copilot 主要在 IDE 中提供代码补全。Claude Code 的 200K 上下文窗口和深度推理能力使其更适合复杂的编程任务。
在中国大陆可以使用吗？
可以。通过 cc-connect 的第三方 API 接入（如 MiniMax M2.7、Kimi K2.6、DashScope GLM-5.1），无需翻墙即可使用。SkillHub 国内镜像提供高速 Skills 下载。同时支持微信接入，手机上也能用。
