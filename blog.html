从零搭建个人导航站 — 纯原生 HTML/CSS/JS

前端 阅读 5 分钟

最近用纯原生技术栈打造了一个个人项目导航站。没有框架、没有构建工具、没有任何依赖——一个 HTML 文件搞定一切。

为什么选择纯原生？

对于个人小站点来说，React/Vue 等框架带来的复杂性往往超过了它们解决的问题。index.html 拖到浏览器就能跑，部署到 Cloudflare Pages 零配置，性能天然极致。

最好的工具不是最流行的那个，而是最能解决你问题的那个。有时候，一个 HTML 文件就够了。

关键特性

项目卡片支持 3D 倾斜效果、光泽鼠标追踪、分类过滤、实时搜索。所有动画使用 CSS transition + requestAnimationFrame，流畅运行在 60fps。

Minecraft Fabric 模组开发入门指南

Minecraft 阅读 8 分钟

Minecraft 模组开发是一个很有趣的领域。Fabric 作为轻量级模组加载器，相比 Forge 有着更快的迭代速度和更好的代码可读性。

环境搭建

首先需要安装 JDK 17、IntelliJ IDEA，然后使用 Fabric 提供的模板生成器创建项目。整个过程大约 10 分钟就能跑起第一个"Hello World"模组。

第一个物品

创建一个自定义物品只需要三个步骤：定义物品实例、注册到游戏注册表、添加纹理资源和本地化字符串。Fabric 的 API 设计非常直观。

建议从简单的物品和方块开始，逐步深入到 TileEntity、网络包、自定义 UI 等高级特性。

玩转 Electron — 桌面应用开发实践

Electron 阅读 6 分钟

用 Web 技术栈开发桌面应用——这就是 Electron 的魅力。最近做了一个可视化 HTML 编辑器，记录一下实践心得。

主进程 vs 渲染进程

Electron 应用分为两个部分：主进程（Node.js）负责窗口管理、系统调用；渲染进程（Chromium）负责界面展示。两者通过 ipcMain / ipcRenderer 通信。

安全注意事项

务必启用 contextIsolation，使用 preload.js 暴露有限的 API 给渲染进程，避免直接使用 nodeIntegration: true。这些是 Electron 官方强烈推荐的安全实践。

Cloudflare Pages 部署个人站点完全指南

部署 阅读 4 分钟

Cloudflare Pages 是我目前最推荐的静态站点托管方案：免费、全球 CDN、自动 HTTPS、Git 集成自动部署。

三步上线

① 把代码推到 GitHub 仓库 → ② 在 Cloudflare Pages 连接仓库 → ③ 设置构建命令（纯静态的话留空即可）。一分钟后你的站点就上线了。

自定义域名

Cloudflare 同时提供域名注册和 DNS 管理，配合 Pages 使用体验丝滑。支持免费 SSL 证书，自动续期，零维护成本。

从 git push 到全球 CDN 分发，只需要 60 秒。这就是现代部署的魅力。
